    Celery Beat

    Perfect — if you're using Celery, then Celery Beat is the cleanest and most integrated way to schedule periodic tasks like retries.

    ✅ What You’ll Set Up

    • Celery – Task runner
    • Celery Beat – Scheduler for periodic tasks (like retry every 10 mins)
    • retry_pending_tasks will become a Celery task
    • Celery Beat will run it every 10 minutes

    🔧 Step-by-Step Setup

    ✅ 1. Install celery[redis] and celery[beat]:

    pip install celery[redis] redis

    ✅ 2. Update celery_worker.py

    from celery import Celery
import time
from models import TaskModel, get_session
import json

app = Celery("worker", broker="redis://localhost:6379/0", backend="redis://localhost:6379/0")

@app.task
def my_task(data):
    print(f"Processing: {data}")
    time.sleep(2)
    return {"result": "done", "input": data}

@app.task
def retry_pending_tasks():
    session = get_session()
    tasks = session.query(TaskModel).filter(TaskModel.status == "pending").all()

    for task in tasks:
        if task.retry_count >= 5:
            continue

        try:
            payload = json.loads(task.payload)
            result = my_task.apply_async(args=[payload])
            task.status = "queued"
            task.celery_task_id = result.id
        except Exception as e:
            task.retry_count += 1
            task.last_error = str(e)
        finally:
            session.add(task)
            session.commit()

    ✅ 3. Create celeryconfig.py (for Beat settings)

    from celery.schedules import crontab

beat_schedule = {
    "retry-failed-tasks-every-10-minutes": {
        "task": "celery_worker.retry_pending_tasks",
        "schedule": crontab(minute="*/10"),
    },
}

broker_url = "redis://localhost:6379/0"
result_backend = "redis://localhost:6379/0"
timezone = "UTC"

    ✅ 4. Update celery_worker.py to load the config

    At the top of celery_worker.py, add this line:

    app.config_from_object("celeryconfig")

    ✅ 5. Run the services

    Start Celery worker:

    celery -A celery_worker.app worker --loglevel=info

    Start Celery Beat:

    celery -A celery_worker.app beat --loglevel=info

    You’ll see: retry-failed-tasks-every-10-minutes show up in the logs every 10 mins 🎯

    ✅ Bonus: Monitor Scheduled Tasks

    You can also use Flower (Celery dashboard):

    pip install flower
celery -A celery_worker.app flower

    Then visit http://localhost:5555
