    Duplicate Check

    Config.toml

    [finacle.dedupcheckcif]
finacle_url = "http://bank_url.com/fiwebservice/services/FIPWebService"
input_file_path = "finacle/DedupCheckCIF.xml"
mime_type = "text"

    Dummy data

    just run api finacle config.toml dedupcheckcif dummy inputdata.toml

# with paramater
just run api config_channel_type="finacle" config_file_path="config.toml" config_entry_name="dedupcheckcif" config_source_data="dummy"

    Database data

    just run api finacle config.toml dedupcheckcif database 12314

# with paramater
just run api config_channel_type="finacle" config_file_path="config.toml" config_entry_name="dedupcheckcif" config_source_data="database" application_id=12314
